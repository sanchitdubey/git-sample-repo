In default case when nothing is mention in script tag. HTML parsing will be continue and when script is found it will stop html parsing and load and execute script first. After that again do the HTML parsing
When Asyn is mentioned in the script tag in that case HTML parsing will be continue when it found script tag load script asyncroniously with html parsing. After loading script Execute the script first and afer that again do the HTML parcing
When Defer is mentioned in the script tag in that case HTML parsing will be continue when it found script tag load script asyncroniously with html parsing. After completing html parsing will execute the script.
